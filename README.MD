# Weather Hourly Bot
Telegram-бот для автоматической рассылки погоды каждый час. 
Позволяет пользователям выбирать свой город.

## Запуск

### 1. Клонируйте репозиторий
```bash
git clone https://github.com/ProstoCyber/WeatherBot.git
cd WeatherBot
```

### 2. Настройте переменные окружения
Создайте файл `.env` в корне проекта, скопируйте в него содержимое файла `.env.example` и замените значения переменных своими.

### 3. Запуск через Docker
Одной командой:
```bash
docker-compose up -d --build
```

## Функционал бота
### /start
Записывает в БД пользователя, если его там нет, и показывает, какой город выбран.

### /setcity [city]
Изменяет выбранный город, изначально стоит город "Москва". Наример, /setcity Токио изменит выбранный город на Токио

### /now
Показывает погоду в выбранном городе в данный момент

## Структура проекта
### weatherbot.py: 
Входная точка, инициализация бота, роутеров и планировщика.

### database.py:
Описание моделей SQLAlchemy и асинхронных методов работы с PostgreSQL.

### Dockerfile:
Инструкция по сборке образа приложения.

### docker-compose.yml:
Описание сервисов контейнеров(бот + база данных) и их связей.

### .env:
Конфиденциальные переменные.
